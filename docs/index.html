Κράτα αναμνήσεις. Παράδωσέ τες στο μέλλον.

Το Memora είναι μια ψηφιακή κάψουλα κληρονομιάς. Δημιουργείς συλλογές αναμνήσεων (φωτογραφίες, βίντεο, φωνή, ιστορίες) και ορίζεις πότε και σε ποιον θα αποκαλυφθούν.

Ζήτα πρόσβαση Δες τι το κάνει μοναδικό
Private-by-design Time-lock memories Family heritage
Τι είναι μια Capsule;
  • Entries: media + αφήγηση
  • Recipients: άνθρωποι που επιλέγεις
  • Reveal: τώρα ή στο μέλλον
  • Export: σφραγισμένο αρχείο μνήμης
